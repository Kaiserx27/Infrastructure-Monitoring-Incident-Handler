# Infrastructure Monitoring & Incident Handler

Python-based tool designed to support Service Desk and NOC teams
in monitoring network and server infrastructure.

## Features
- Network availability monitoring (ping, ports)
- Web service health checks (HTTP)
- Automatic incident creation
- Incident storage in SQLite database
- Service restart actions
- Logging and alert handling

## Use Case
Tool can be used by 1st line support to monitor systems,
analyze alerts and perform first-level remediation
according to operational procedures.

## Technologies
Python 3, Linux, SQLite, networking, monitoring
